
# Fuel Cost Calculator

A simple Python application that calculates fuel consumption and travel cost based on the user's input.

## Features

* Calculate the fuel required for a trip
* Calculate the total fuel cost
* Calculate round-trip fuel consumption and cost
* Validate negative input values

## Technologies

* Python

## How to Run

```bash
python main.py
```

## Example

**Input**

* Distance: 350 km
* Fuel Consumption: 6.5 L/100 km
* Fuel Price: 58 TL/L

**Output**

* Fuel Needed: 22.75 L
* Total Cost: 1319.50 TL

## About

This project is part of my Python learning journey. The goal was to practice functions, parameters, return values, user input, and basic calculations while building a small real-world application.
